
## Estrutura das Pastas

* **`include/`**: Contém os arquivos de cabeçalho (`.h`).
  * **`data_structures/`**: Cabeçalhos das estruturas de dados.
  * **`algorithms/`**: Cabeçalhos dos algoritmos.
* **`src/`**: Contém os arquivos de código-fonte (`.c`).
  * **`data_structures/`**: Implementação das estruturas de dados.
  * **`algorithms/`**: Implementação dos algoritmos.
* **`main.c`**: Arquivo principal utilizado para testar e executar os algoritmos implementados.

## Compilar e executar

### Compilação

Para compilar o projeto, execute o comando na raiz do repositório (dsa-in-c):

```bash
make
```

### Limpeza

Para apagar os arquivos objeto intermediários (`.o`) e o executável, execute:

```bash
make clean
```
