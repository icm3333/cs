## Estrutura das Pastas

* **`include/`**: Contém os arquivos de cabeçalho (`.h`).
* **`src/`**: Contém os arquivos de código-fonte (`.c`).
* **`main.c`**: Arquivo principal utilizado para testar e executar os algoritmos implementados.

## Compilar e executar

### Compilação

* **`No Linux`**: execute o comando make

* **`No Windows (CMD / PowerShell via MinGW)`**: execute mingw32-make


### Limpeza
Para apagar os arquivos objeto intermediários (.o) e o executável execute:

* **`No Linux`**: make clean

* **`No Linux`**: mingw32-make clean
